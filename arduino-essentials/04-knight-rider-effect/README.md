# Knight Rider LED Chaser

This project demonstrates a sequential LED animation using an Arduino Uno. The code is written in C++ using arrays.

# Demo
![Knight Rider Animation](KnightRiderEffectAnimation.gif)

# Components
- Arduino Uno
- 8x LEDs
- 8x 330Ω Resistors
- Jumper Wires & Breadboard

# How to Use
1. Connect the LEDs to digital pins 2 through 9.
2. Open `KnightRiderEffect.ino` in Arduino IDE.
3. Upload the code to your board.
